# Bank Account Simulation

## Description
This is a Java OOP project that simulates a basic bank account system. It allows users to perform banking operations such as deposit, withdrawal, balance checking, and viewing transaction history.

## Features
- Deposit Money
- Withdraw Money
- Check Account Balance
- View Transaction History
- Menu-driven Console Application
- Object-Oriented Programming Concepts

## Technologies Used
- Java
- VS Code / IntelliJ IDEA

## Project Structure
```
BankAccount.java
Transaction.java
Main.java
```

## How to Run
1. Clone the repository.
2. Open the project in VS Code or IntelliJ IDEA.
3. Compile all Java files.
4. Run `Main.java`.
5. Follow the on-screen menu.

## Sample Operations
- Deposit Money
- Withdraw Money
- Check Balance
- View Transaction History
